# quant-data

Data collectors for quant. 数据收集、清洗和存储等功能。

高质量的数据是量化的立足之本，谨防“Garbage in, garbage out”。

项目roadmap、技术cheatsheet和API文档：[quant-data WIKI](https://github.com/X0Leon/quant-data/wiki)。

Dependencies:

* python 3/2.7
* pandas
* tushare
* PyMySQL
